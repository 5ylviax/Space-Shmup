# Space Shmup Game 
*Unity Game Development*
A 2D space shooter built in Unity, inspired by classic arcade games.  
Features custom enemy, power-up systems, scoring UI, and integrated sound effects.
## Enemies Implemented 
- Enemy_0: Base enemy inherited from Enemy class
- Enemy_1: Moves in a sine-wave pattern
- Enemy_2: Moves with linear interpolation + easing
- Enemy_3: Follows Bézier curve path
- Enemy_4: Includes shield protection that reduces damage
## Weapons & Power Ups 
- none 
- blaster 
- spread 
- shield 
## Enhancements 
- Score Tracker - displays player progress
- Shooting Audio - sound effect when firing weapons

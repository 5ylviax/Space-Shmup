# Space Shmup Game 
*Unity Game Development*
A 2D space shooter built in Unity, inspired by classic arcade games.  
Features custom enemy, power-up systems, scoring UI, and integrated sound effects.

Link to Game: https://play.unity.com/en/games/9c4f996c-652d-4b03-9027-d425ae417875/space-shump-cs-382
